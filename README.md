A [Copier][1] template for Python applications with [pywebview][2].

## Quickstart

1. Install the latest Copier via command:

    ```text
    pip install --upgrade copier copier-template-extensions
    ```

2. Use command `copier` to generate a new project:

    ```text
    copier copy gh:garrett-he/copier-python-pywebview /path/to/destination
    ```
